El siguiente "Aviso de privacidad" explica nuestras prácticas de datos en relación con el Servicio, y solo se aplica a las prácticas de datos con respecto al Servicio y no a nuestro sitio web, a ninguno de nuestros otros productos o servicios, o cuando nos contacta por correo o teléfono para servicio al cliente o de lo contrario. Cómo funciona el servicio

Cuando utiliza el navegador web Viasat y navega hacia un sitio web, o cuando el navegador predice que puede navegar a un sitio web, el navegador envía la URL del sitio web que está tratando de visitar, o se prevé que visitará, al Servicio de Internet de Viasat. . A continuación, el servicio de asistencia por Internet de Viasat proporciona al navegador web Viasat sugerencias sobre cómo renderizar el sitio más rápido. Cuando el sitio termina de cargarse, el navegador envía información al Servicio de asistencia por Internet de Viasat que describe el proceso de carga para que el sistema de sugerencias pueda aprender y enviar mejores pistas la próxima vez. 

¿Qué información recopilamos y cómo puede limitar la recolección?

Recopilamos datos para operar y mejorar el Servicio. Por ejemplo, recopilamos datos para que el Servicio de orientación de Viasat aprenda continuamente y brinde mejores y mejores sugerencias, lo que finalmente ayuda a habilitar cargas de página más rápidas y más rápidas. Para ello, el navegador web Viasat comparte datos con el servicio de sugerencias de Internet de Viasat con respecto a qué objetos (p. Ej., URL completas) eran necesarios para representar la página, la línea de tiempo asociada a la obtención y uso de los objetos y la información de respaldo, como si el objeto en el caché del navegador y si el dominio estaba en el caché de DNS. Además, Viasat Web Browser comparte datos con Viasat Internet Hinting Service con respecto a la información sobre el entorno de navegación, como si el bloqueo de anuncios está habilitado e información sobre la CPU del dispositivo, el disco,

También podemos recopilar información adicional relacionada con el rendimiento de su conexión a Internet para ayudar a solucionar problemas y optimizar su Servicio. Si está navegando en el servicio de Internet Exede o WildBlue de Viasat, las prácticas de recopilación de datos de Viasat se describen en la Política de privacidad para ese servicio en particular. Puede ver la Política de Privacidad del Suscriptor Exede aquí , y la política de privacidad de abonado WildBlue aquí . 

Si el Servicio falla, recibiremos informes de fallas que incluyen datos relacionados con ese incidente, que pueden incluir información que lo identifica personalmente (" Información de identificación personal " o " PII ") (por ejemplo, cómo ha nombrado su dispositivo), que solo usamos en relación con la evaluación del mal funcionamiento y los esfuerzos para corregir el problema y mejorar nuestros productos y servicios. Además, tiene la opción de enviarnos informes de comentarios que incluyan datos relacionados con el uso de su Servicio, y al hacerlo, acepta que usemos esos datos para mejorar nuestros productos y servicios.

El navegador web Viasat no identifica y comparte información con respecto a sus tarjetas de crédito, contraseñas, números de seguro social ni a ningún otro dato ingresado por el usuario ni a ningún otro dato que se le entregue con el servicio Viasat Hinting. ¿Cómo aseguramos su información?

Ningún servicio de Internet puede ser totalmente seguro. Sin embargo, tomamos medidas razonables para proteger sus datos. Por ejemplo, la seguridad del servicio Viasat Hinting está de acuerdo con las normas ISO / IEC 27001 para sistemas de gestión de seguridad de la información y 800-171 / 800-122 para proteger la información controlada. Todas las comunicaciones que contienen PII entre Viasat Web Browser y Viasat Internet Hinting Service están encriptadas de extremo a extremo con Transport Layer Security (TLS).

Además, Viasat Web Browser es compatible con "Navegación segura", un servicio de Google que permite a las aplicaciones verificar las URL contra las listas constantemente actualizadas de Google de sospechosos de phishing, malware y páginas de software no deseadas. Todos los días, la tecnología de Navegación segura de Google examina miles de millones de URL y descubre miles de nuevos sitios inseguros, muchos de los cuales son sitios web legítimos que se han visto comprometidos. Cuando Google detecta sitios inseguros, el navegador web Viasat muestra una advertencia. El servicio de Navegación segura de Google es un Servicio de terceros y no somos responsables de su eficacia ni de sus datos o prácticas comerciales. Consulte aquí para obtener más información sobre el servicio de navegación segura de Google.

¿Cómo puede ayudar a proteger su privacidad y proteger su información?

El navegador web Viasat ofrece servicios de bloqueo que lo ayudan a protegerlo contra anuncios inseguros, malware y rastreo de privacidad a través de características que pueden habilitarse o deshabilitarse en la instalación inicial del navegador. Si está utilizando la versión de escritorio del navegador web Viasat, durante la instalación, se le preguntará si desea habilitar o rechazar un servicio de bloqueo de anuncios. Si opta por el servicio de bloqueo de anuncios, se habilitará una extensión de terceros llamada uBlock Origin que bloquea los anuncios, los rastreadores de privacidad y el malware para ayudar a mejorar la seguridad y la velocidad de carga de la página. uBlock Origin es un Servicio de Terceros y no somos responsables de su eficacia, ni de sus datos o prácticas comerciales. Mira aquípara más información sobre uBlock Origin. Si está utilizando la versión móvil del navegador web Viasat (para dispositivos iOS y Android), un anuncio desarrollado por Viasat, rastreador de privacidad y bloqueador de malware está habilitado por defecto. Puede cancelar esta característica por completo o deshabilitarla de manera selectiva para ciertos sitios web navegando al menú de configuración.
